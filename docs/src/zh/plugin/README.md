obsidian 的生态非常完善，基于爱好者的开发热情，有着800多个插件已经上架**社区插件市场**。而且还不断有着新的开发者为其开发新的插件。当然，还有一部分没有上架的插件也可以被安装使用。

我们来认识一下obsidian的插件：
- 核心插件，是obsidian自带的插件，无需安装开启就可以使用。
- 社区插件，需要打开安全模式，自行在社区插件市场安装。目前在国内已经能够正常打开
