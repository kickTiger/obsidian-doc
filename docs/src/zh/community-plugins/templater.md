**插件简介：**  
templater，obsidian 必装插件之一，提供了固定模板和动态模板。提升obsidian使用效率，可以按照模板新建文件，节省大量时间。

::: warning
推荐和 quickadd 插件以及 button 插件联合使用，效果更棒
:::

**插件安装：**
::: tip 插件安装
打开 obsidian → 设置 ⚙️ → 第三方插件 → 社区插件市场，搜索关键字安装

注意：你需要关闭第三方插件的**安全模式**，才能安装社区插件，建议关闭。
:::
