**说明**  
也是应群友提问，是否能够达到批注的效果。其实也是可以的。

**演示**  
将鼠标移动到咖啡豆上，会看到批注的提示。在 obsidian 中也是这样的效果。

::: info 批注效果
我们<u title="游戏狂">咖啡豆</u>喜欢的obsidian
:::

**语法**  
```html
我们<u title="游戏狂">咖啡豆</u>喜欢的obsidian
```
