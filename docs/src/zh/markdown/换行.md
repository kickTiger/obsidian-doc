**语法：**
在markdown中换行，有一些方法，因为各家的markdown编辑器不同，所有也会有不同的呈现，可以自己尝试。

```markdown
在当前行的结尾加 2 个空格··
这行就会新起一行\
反斜杠也可以换行
```

**使用代码换行**

HTML代码也是被支持的。可以使用如下代码换行，一个表示一行。
```html
<br>
```

**示例：**  
我是咖啡豆我要开始换行了  
第二行
```markdown
我是咖啡豆我要开始换行了  
第二行
```

::: tip 自定义标题
上方的代码中 `换行了` 后面有两个空格
:::
