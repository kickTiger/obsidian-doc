::: tip 下图中的演示使用说明
1. 画布中的登陆不用管，和网站不是一个体系
2. 拖动画布：图中按住鼠标右键可拖动画布
3. 放大画布：右下角 百分比 点击可放大，箭头可选择缩放比
4. 快捷放大：按住Ctrl，滚动鼠标滑轮，可自由缩放大小
5. 快捷导航：左下角图标(路线图列表)，页面内快速跳转
:::
