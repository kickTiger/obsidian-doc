**说明：**  
实现通过 dataview 查询指定文件夹下的所有文件

**使用方法：**  
1. 复制以下代码，在任意md文件中插入即可。
2. 注意修改文件路径为自己库中的路径，否则会报错

::: tip 添加dataview查询的方法
1. 用代码块包裹查询代码（第一行注明语言 `dataview`和最后一行对应结尾）
2. 任意位置插入即可，注意修改文件路径和查询条件
:::

**代码**  
代码框右下有复制按钮，点击自动复制代码
````markdown
```dataview
list
from "文件夹名称"
sort file.ctime desc
```
````
